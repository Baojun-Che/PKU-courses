\section{实验概述}
本实验实现了稀疏矩阵CSR格式的存储结构，并基于此实现了Jacobi迭代法求解二维Poisson方程.通过不同网格尺寸的数值实验，分析了Jacobi迭代法的收敛性能和计算效率.

\section{代码结构}
实验代码采用模块化设计，主要文件结构如下：

\begin{alltt}
sparse_matrix/
├── .vscode/              # VSCode配置目录
│   ├── launch.json       # 调试配置
│   ├── settings.json     # 编辑器设置
│   └── tasks.json        # 编译任务
├── include/              # 头文件目录
│   ├── solvers.hpp       # 求解器声明
│   └── sparse_matrix.hpp # 稀疏矩阵类声明
├── src/                  # 源文件目录
│   ├── solvers.cpp       # 迭代法实现
│   └── sparse_matrix.cpp # 稀疏矩阵类实现
└── test/                 # 测试目录
    └── test.cpp          # 性能测试代码
\end{alltt}

\subsection{核心模块说明}
\begin{itemize}
    \item \textbf{sparse\_matrix.hpp/cpp}: 实现了CSR格式的稀疏矩阵类，支持矩阵向量乘法、三角部分提取等操作
    \item \textbf{solvers.hpp/cpp}: 实现了Jacobi迭代法和向量范数计算
    \item \textbf{test.cpp}: 包含Poisson矩阵生成器和性能测试框架
\end{itemize}

\section{结论与展望}
本实验成功实现了基于CSR格式的稀疏矩阵类和Jacobi迭代求解器，并测试了不同规模Poisson方程的求解性能.结果表明：
CSR格式能有效存储和操作大型稀疏矩阵; Jacobi迭代法对于Poisson方程是可行的，但收敛速度较慢, 未来将实现Gauss-Seidel、SOR等收敛更快的迭代方法.
